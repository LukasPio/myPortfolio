Buscando constantemente conhecimento, adquiri habilidades em diversas linguagens e frameworks, permitindo-me criar websites e aplicações web funcionais e interativas. Tecnologias que domino:

HTML
HTML é uma linguagem de marcação usada para criar a estrutura de páginas da web, organizando elementos como textos, imagens e links.
CSS
CSS é uma linguagem de estilo que formata a aparência dos elementos em uma página web, permitindo definição de cores, fontes e posicionamento.
JavaScript
JavaScript é uma linguagem de programação que adiciona interatividade a páginas web, possibilitando animações e manipulação de elementos.
Adobe PhotoShop
Adobe Photoshop é uma ferramenta de edição de imagens amplamente usada por designers, permitindo criar, editar e manipular imagens digitais.
Office 365
Office 365 é uma suíte de aplicativos Microsoft, incluindo Word, Excel e PowerPoint, utilizada para criação e edição de documentos.
Git / GitHub
Git é um sistema de controle de versão para rastrear alterações em projetos, e o GitHub facilita o compartilhamento e colaboração entre desenvolvedores.

Projetos principais:

Jogo da velha

Uma aplicação interativa do clássico jogo da velha. Importante para praticar lógica de programação, interatividade e manipulação do DOM para criar uma experiência de jogo envolvente.

Calculadora com JavaScript

Uma aplicação que permite aos usuários realizar operações matemáticas básicas. Importante para praticar manipulação de eventos, lógica de programação e manipulação do DOM.

Contador regressivo

Um projeto que exibe o tempo restante até uma data e hora específicas definidas pelo usuário. Importante para praticar manipulação de datas, temporizadores e atualização em tempo real.
